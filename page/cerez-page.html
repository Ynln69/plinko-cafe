Çerez Politikası

Yürürlüğe girme tarihi: 1 Ocak 2024

PlinkoCoffee.com, web sitemizdeki deneyiminizi geliştirmek ve size daha kişiselleştirilmiş hizmetler sunmak için çerezleri kullanır.

1. Çerezler nedir:

Çerezler, bir web sitesinin cihazınıza kaydettiği küçük dosyalardır. Bu dosyalar, web sitesinin işleyişini iyileştirmek ve kullanıcı deneyimini optimize etmek için kullanılır.

2. Çerezleri nasıl kullanıyoruz:

  • Web sitesinin performansını izlemek ve deneyiminizi geliştirmek için.
  • Kullanıcı ayarlarınızı hatırlayarak, size daha iyi hizmet sunmak için.

3. Kullandığımız çerez türleri:

  • Gerekli çerezler: Web sitesinin temel işlevselliği için gereklidir. Örneğin, alışveriş sepetinizi hatırlamak veya giriş bilgilerinizi korumak için kullanılır.
  • Analitik çerezler: Web sitesinin kullanımını izlemek ve performansını artırmak için kullanılır. Bu çerezler, web sitemizin nasıl kullanıldığını anlamamıza yardımcı olur.

4. Çerez yönetimi:

Tarayıcı ayarlarınızdan çerezleri yönetebilir veya devre dışı bırakabilirsiniz. Ancak, çerezleri devre dışı bırakmanız durumunda web sitemizin bazı özellikleri düzgün çalışmayabilir.

Web sitemizi kullanarak, bu politikada açıklanan çerezlerin kullanımını kabul etmiş olursunuz. Çerezlerle ilgili herhangi bir sorunuz varsa, lütfen support@plinkocoffee.com adresinden bizimle iletişime geçin.
